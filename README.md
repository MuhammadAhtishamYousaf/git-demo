# Git


`git clone` : To clone a directory(repo)
`cd` : To change directory
`ls` : To list visible files
`ls -a` : To list all (include hidden) files
`git status` : To check the current code status
    untrcked : new files taht git doesn't yet track
    modified : changed
    stages : file is ready to be committed
    unmodified : unchanged

`git add <filename>` : add new or changed files in your working directory to the git staging area.
`git commit -m "some message"` : it is the record of change
`git push origin main` : To upload local repo content to remote repo
