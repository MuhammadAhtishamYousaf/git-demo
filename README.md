# Git <br>
`git clone` : To clone a directory(repo) <br>
`cd` : To change directory <br>
`ls` : To list visible files <br>
`ls -a` : To list all (include hidden) files <br>
`git status` : To check the current code status <br>
    `untrcked` : new files taht git doesn't yet track
    `modified` : changed
    `stages` : file is ready to be committed
    `unmodified` : unchanged

`git add <filename>` : add new or changed files in your working directory to the git staging area. <br>
`git commit -m "some message"` : it is the record of change <br>
`git push origin main` : To upload local repo content to remote repo <br>
